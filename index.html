The Backbone of Modern Website

Implementation of HTML, CSS dan JavaScript

Introduction

Di era digital ini peran sebuah website menjadi kebutuhan penting bagi setiap kebutuhan, baik dalam keperluan bisnis seperti pemasaran,penjualan barang dan jasa, penyampaian informasi, dan masih banyak lagi, Sehingga peran website sangatlah penting.Untuk membuat dan mengembangkan website digunakanlah 3 bahasa pemerograman utama ya itu Hypertext Markup Language (HTML), Cascading Style Sheet (CSS), dan Javascript sebagai penunjang utama pembuatan website.

History

Sejarah Website pertama kali diluncurkan tahun 1991, webiste tersebut bernama info.cern.chyang diperkenalkan oleh ilmuan dari Cern Swiss yang bernama Tim Berners-Lee melalui World Wide Web (WWW). pada awal diperkenalkan nya, website hanya terdiri dari text sederhana, namum seiiring dengan perkembangan teknologi, diperkenalkan lah CSS dan Javascript sehingga pada website modern tidak hanya menampilkan text, namun juga berbagai media seperti gambar, video serta terdapat berbagai macam animasi dan transisi serta di perkenlakan juga website responsive yang dapat menyesuaikan tampilan dengan ukuran perangkat.

Implementation

What is HTML ?

HTML Adalah Hypertext Markup Language yang merupakan struktur dasar dalam pembuatan website modern. Sejak peluncuran website pertama di dunia oleh ilmuan asal Swiss hingga saat ini versi versi HTML terus di kembangkan, dimuali dari versi HTML 1.0 yang hanya terdapat 18 tag seperti tag p untuk paragraf, a untuk link, img untuk menampilkan gambar, dan h yang terdiri dari h1 hingga h6 untuk menentukan ukuran text, hingga versi terbaru saat ini yaitu versi HTML 5.2 yang telah mendukung fitur fitur modern seperti penambahan dukungan untuk multimedia, hingga tersedia nya fitur grafik dan animasi.

HTML Structure

HTML terdiri dari elemen elemen penyusun yang disebut juga dengan tag html, penggunaan tag html diawali dengan tag pembuka dan diakhiri pula dengan tag penutup, pada umum nya html tersusun dari tag doctype, jenis bahasa, head, dan body. tag doctype bertujuan untuk mendifinisikan jenis file menjadi jenis HTML, tag bahasa digunakan untuk mendefinisikan bahasa yang ingin diginakan dalam struktur html, tag head bertujuan sebagai identitas dari HTML, tag head umum nya berisi judul website tipe karakter set yang menggunakan UTF-8 serta tag meta yang bertujuan untuk menyesuaikan ukuran website ke device pengguna.

Selanjutnya tag utama dalam HTML yaitu tag Body, yang merupakan tempat untuk meletakan isi content dari website. Dalam tag ini terdapat banyak sekali tag yang menunjang keperluan dari website tersebut dimulai dari tag p yang berfungsi untuk menampilkan text hingga text img yang dapat menampilkan media seperti gambar dan masih banyak lagi. dengan HTML 5 versi terbaru terdapat tag iframe yang berguna untuk menampilkan kontent dari website lain dengan demikian user dapat menempatkan potongan maps yang telah di sediakan oleh google maps.

Cascading Style Sheet

What is CSS ?

CSS adalah Cascading Style Sheet yang berutujuan untuk menambahkan desain ke dalam website, tentu saja website yang hanya berisi text akan membosankan dan kurang menarik untuk di buat, oleh karna itu digunakanlah CSS untuk memperbaiki tampilan dari website. CSS pertama kali rilis pada tanggal 17 Desember 1996 dengan versi CSS1 yang memiliki fitur utama berupa color, font dan text-align hinga pada saat ini CSS telah berkembang menjadi CSS 3 yang mendukung media query sehingga memungkinkan untuk pembuatan website secara responsive, serta terdapat dukungan untuk animasi dan transisi

How Do You Use CSS

CSS dapat digunakan dengan beberapa cara yaitu secara inline, internal dan juga eksternal.Penggunaan CSS secara Inline yaitu menempatkan css pada setiap tag HTML dengan cara menggunakan style yang diikuti dengan pemanggilan tag CSS yang ingin digunakan. Penggunakan CSS secara inline yaitu tetap menempatkan CSS pada file HTML namum penemempatan nya terletak pada body dan diawali dengan tag style, berbeda dengan inline CSS untuk menerapkan CSS pada tag html tertentu maka digunakan lah id dan class pada tag HTML tersebut. kemudian penggunaan CSS external yaitu menempatkan CSS pada file terpisah dengan file HTML sehingga mempersingkat kode di dalam satu file.

CSS Structure

Struktur dari CSS tidak jauh berbeda dari struktur HTML, dimana tag yang akan di beri CSS di panggil dengan mendahului titik untuk pemanggilan class dan pagar (#) untuk memanggil id gari tag tertentu setelah itu diikuti dengan kurung kurawal dan isi dari berbagai tag CSS yang akan digunakan.

CSS Framework

Untuk memudahkan user dalam menggunakan CSS maka dibuatlah Framework CSS, framework css adalah sebuah kerangka yang telah di kembangkan oleh pihak tertentu, sehingga dalam penggunaan nya user hanya perlu memanggil class yang telah ada dengan penggunaan framework ini memungkinkan untuk hanya memilki 1 file html, pada saat ini terdapat banyak sekali pihak yang menyediakan framework framework CSS sehingga para developer dapat memilihi sesuai dengan keiinginan dan kebutuhan website yang dikembangkan.

Tailwind CSS

Tailwind CSS merupakan tipe framework css yang menerapkan utility-fisrt Framework, yaitu menyediakan utilitas utilitas kecil yang dapat lnagusng di implementasikan kedalam tag html dengan cara memanggilnya ke dalam class yang ada di tag html

Boostrap

Merupakan framework css yang sangat populer digunakan karena simplisitasnya, Boostrap menyediakan komponen komponen siap pakai sehingga para developer hanya perlu memanggilnya saja ke dalam file HTML

UIKIT

UIKit merupakan framework CSS yang dikembangkan menggunakan pendekatan mobile-first, dengan demikian UIKIT dapat menyesuaikan diri dengan setiap perangkat baik komputer maupun handphone. UIKIT juga mendukung fitur ui yang efisien dan responsive

JavaScript

What is JavaScript ?

JavaScript adalah bahasa pemerograman berorientasi object (PBO) yang digunakan untuk mendukung fungsionalitas website, Javascript menawarkan banyak sekali fitur diantara nya dalah fitur interativitas yang dinamis, dimana pengguna dapat berinteraksi dengan website tanpa harus memuat ulang, selain itu juga Javascript merubah struktur HTML dan CSS sehingga website dapat berubah dan menyesuaikan, selain itu juga Javascript dapat digunakan untuk menambahkan animasi dan transisi pada website, sehingga website tersebut akan terlihat lebih dinamis

How Do You Use JavaScript

Penggunaan Javascript sama hal nya dengan penggunaan CSS dimana terdapat 2 cara penemempatannya yaitu internal dan external. Penggunaan secara internal yaitu meletakan script di dalam file HTML dengan menggunakan tag script diikuti dengan penulisan Javascript. Contoh syntax Javascript di samping merupakan contoh penerapan Javascript secara external dimana syntax Javascript diletakan pada file terpisah dan di panggil ke dalam file html melalui link yang terletak pada tag head.

Why you should Use JavaScript

Seiring berkembangnya dunia Digital peran Javascript sangat penting dalam suatu website, yang dimana penambahan animasi dan transisi tidak hanya mempercantik sebuah website namun juga menjadi daya tarik agar website tersebut banyak dikunjungi oleh pengguna internet, selain itu dengan pengimplementasian JavaScript akan membuat website menjadi interaktive sehingga lebih memudahkan para pengguna untuk memahami informasi yang ditampilkan pada website tersebut.

Javascript Framework

Javascript Juga memiliki framework framework yang dapat mempermudah dalam pengimplementasiannya, framework JS berisi struktur struktur dasar dan juga template sehingga dalam pengembangan nya developer tidak perlu membuat struktur Javascript dari awal, terdapat banyak pilihan framwork Javascript yang dapat digunakan berdasarkan keperluan dan juga keinginan para developer, 3 contoh diantara nya adalah React JS, Vue Js dan Node JS.

React Js Framework

React Js merupakan framework Javascript yang dikembangkan oleh Facebook untuk menunjang fitur antarmuka pengguna, salah satu keunggulan dari react Js adalah sifatnya yang sangat fleksibel sehingga cocok untuk semua penggunaan.React Js juga mendukung fitur Virtual DOM sehingga react hanya memperbaruhi bagian yang berubah saja tanpa memperbaharui semua halaman sehingga meningkatkan efisiensi.

Vue Js Framework

Vue Js adalah frameworks JS yang bersifat prograsif yang memberikan fleksibilitas untuk membangun aplikasi dengan antarmuka pengguna yang dinamis, keunggulan yang dimiliki framework ini yaitu Ekosistem yang kuat serta tingkat kinerja yang tinggi dan pendekatan berbasis komponen, menjadikan nya framework yang cocok untuk berbagai jenis kebutuhan.

Node Js Framework

Walaupun bukan merupkan frameworks js yang bersifat antarmuka, Node JS adalah framework js yang dikembangkan untuk sisi server sehingga memungkinkan developer untuk menjalankan Javascript diluar browser.Node Js dikembangkan menggunakan mesin V8 yang dimilki oleh Google sehingga membuatnya sangat efisien dan ringan.
